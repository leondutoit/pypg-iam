
# pypg-iam

Python library for [pg-iam](https://github.com/unioslo/pg-iam).

# Running tests

```bash
# set postgres environment variables for pg-iam db access
export PYPGIAM_USER=""
export PYPGIAM_PW=""
export PYPGIAM_HOST=""
export PYPGIAM_DB=""

# and the run tests
pytest iam/tests.py
```
